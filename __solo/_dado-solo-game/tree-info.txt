NIVEL 0 -> _dado-solo-game
-------

Descripción:
------------
Esta sería la primera implementación de NumbersDice. El código es preferentemente hecho con python.

NIVEL 1 
-------

-> .venv		:: Contiene un entorno de desarrollo de python
-> .vscode		:: Configuraciones de VSC
-> consola		:: la implementación gráfica de numbersdice (consola)
-> kernel		:: piezas físicas del juego
-> ejecutable.py	:: sería el conector de la lógica y lo visual, aparte de que ejecutaría el juego
-> tree-info.txt	:: ARCHIVO ACTUAL

NIVEL 2 :-> "kernel"
-------
-> _importaciones.py	:: contiene todos los módulos importados y las variables globales
-> modos-juego		:: modos de juego
-> piezas.py		:: contiene las piezas móviles del Juego
-> objetos.py		:: contiene a las piezas inmóviles del juego
-> mapa.py		:: contiene los mapas del juego
-> _tablero.py		:: el módulo tablero, creará los tableros del juego
-> valores.py		:: todos los valores que definin parámetros del juego
-> temp_prueba_codigo	:: archivo temporal que se utiliza solo para provar código
